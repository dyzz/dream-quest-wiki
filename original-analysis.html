Dream Quest 原版解析与 Remaster 研究 ​

结论摘要 ​

original/ 是一份 iOS App Store 形态的 Dream Quest 包，不是可直接运行的 Unity 工程。它包含 Unity 4.7.2f1 的 data files、IL2CPP 元数据和 arm64 Mach-O 主程序，但没有原始 UnityScript/C# 源码。

目前已经从原包提取出八类可用证据：

  • Unity asset 索引：extracted/unity_assets_index.json
  • Unity Resources 路径索引：extracted/resource_paths.json
  • Texture2D PNG 导出与资源路径映射：extracted/textures/、extracted/structured/texture_export_manifest.json
  • AudioClip MP3 导出：extracted/audio/、extracted/structured/audio_export_manifest.json
  • Remaster-facing 素材目录：extracted/structured/remaster_asset_catalog.json
  • Starter prototype 素材包：extracted/structured/remaster_starter_asset_pack.json
  • Starter static-hosted browser prototype：prototype/starter/index.html
  • IL2CPP metadata 字符串和标识符：extracted/il2cpp_metadata_index.json、extracted/il2cpp_string_literals.json
  • IL2CPP 类型/字段/方法模型：extracted/il2cpp_types_index.json、extracted/il2cpp_game_model.json、extracted/structured/key_systems.json

关键内容清单已经交叉生成在 extracted/content_inventory.json：

  • 卡牌资源路径 375 个，其中 364 个能匹配到同名 MonoScript 类。
  • 地牢大图 actor/monster 资源路径 99 个，其中 93 个能匹配到同名 MonoScript 类。
  • 职业资源 18 个形态/贴图入口，16 个匹配到职业脚本；其中包括 Druid human/wolf 形态和 Random。
  • Combat ability 图标 32 个，全部匹配到脚本类。
  • Dungeon talent 图标 51 个，其中 14 个直接匹配到 DungeonAction 类；剩余多数是 passive/talent 数据而不是一类一脚本。
  • Texture2D 已导出 1115 个 PNG；2 个 0x0 Font Texture 无图像数据。ResourceManager 直接映射加少量名称 fallback 后，资源路径匹配 1093/1239，核心图像类别（card/combat ability/dungeon actor/profession/talent/achievement）全部有唯一 PNG 候选；剩余 146 个未匹配路径属于 material、prefab 或 font/other resource。
  • AudioClip 已导出 13/13 个 MP3，输出到 extracted/audio/by_container/sharedassets0/，manifest 为 audio_export_manifest.json；这些是环境/菜单音乐和少量音效，不是 Bard/Sing 的 20 首玩法 song 文本目录。
  • remaster_asset_catalog.json 已把内容条目接到 PNG：401 个 card 资源/规则 union、99 个 dungeon actor、18 个 profession、37 个 combat ability 和 51 个 dungeon talent icon；5/5 个素材覆盖检查通过。该目录显式标注 31 个 card-art-only 资源、26 个 structured-only card、26 个非 monster-catalog dungeon actor/feature、2 个 profession form/picker-only 资源、5 个无独立 inventory icon 的 combat ability 和 8 个 icon-only dungeon talent 资源。
  • remaster_starter_asset_pack.json 已把 starter fixture 接到 PNG：10 张 starter card、3 个 starter monster 的 big/tile 图、Warrior big/little 图和 7 个 floor 1 视觉资源，共 21 个条目、25 个 PNG 引用；5/5 个素材包检查通过，所有引用文件均存在。
  • prototype/starter/ 已提供静态托管的浏览器原型，直接加载 starter/full catalog JSON 和导出 PNG，覆盖 Overview/Cards/Professions/Monsters/Dungeon/Remaster/Combat/Play 八个视图；Cards 视图可在 starter 子集和 370 张 recovered cards 全量目录间切换，Monsters 视图可在 starter 子集和 73 个 recovered monsters 全量目录间切换，Dungeon 视图展示 floor-1 seed snapshot 和 recovered generation rule summary，Remaster 视图展示当前 completion audit、阻塞项、artifact/view 覆盖、runtime boundary、area checklist 和下一步工程队列，Professions 视图浏览 16 个职业、37 个 combat ability、33 个 dungeon action 和 43 个 DungeonTalent，Play 视图可跑简化地牢移动、starter card combat、敌方回合、胜利清格和 reset，但仍不是原版准确 runtime。

原包取证 ​

Info.plist 证据：

  • App 名称：Dream Quest
  • Bundle ID：com.Peter.DreamQuest
  • 版本：CFBundleShortVersionString = 1.12
  • 可执行文件：DreamQuest
  • 目标平台：iPhoneOS，landscape

Unity/IL2CPP 证据：

  • original/Data/mainData 内含 Unity 版本字符串：4.7.2f1
  • original/DreamQuest 是 Mach-O 64-bit executable arm64
  • original/Data/Managed/Metadata/global-metadata.dat 头部 magic 为 0xfab11baf，metadata version 为 16
  • original/DreamQuest 的 LC_ENCRYPTION_INFO_64 标记 cryptid = 1，且从 0x7000 开始的第一页仍呈加密/不可反汇编形态；0x8000 之后可见正常 arm64 指令。因此控制流反编译可做，但需要跳过或处理加密页。
  • mainData 中记录了场景：Assets/Scenes/MainMenuScreen.unity、Assets/Scenes/Dungeon.unity
  • 代码主体来自 Assembly-UnityScript.dll，另有少量 Assembly-CSharp.dll、Assembly-CSharp-firstpass.dll、UnityEngine.UI.dll
